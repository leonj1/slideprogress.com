# Slide Progress

Track your presentation topics in real-time. Mark each topic complete as you present, see your progress at a glance.

## Features

- Create multiple presentations with custom topics
- Drag-and-drop topic reordering
- Click topics to mark complete during presentation
- Progress indicator dots
- Data persists in browser localStorage

## Quick Start

1. Open `version6-muted-minimal.html` in your browser
2. Click **+ New** to create a presentation
3. Add topics and save
4. Click the presentation to start presenting

## Deploy

Configured for Netlify. Connect your repo and deploy automatically.

## Tech

Static HTML/CSS/JS. No build step. No dependencies.
